# Lauda — Overview

## TL;DR

**Lauda** — AI SaaS-сервис для инфлюенс-маркетинга на международном рынке. Помогает компаниям увеличивать охваты с помощью искусственного интеллекта. Стадия: early, поиск PMF.

> ⚠ Эта страница — **стартовый каркас**. Заполняется по мере накопления знания через ingest интервью, исследований и стратегических решений. Confidence: medium — здесь зафиксировано то, что заявлено CEO, но не валидировано данными с рынка.

## Что мы делаем

AI-инструмент для брендов и агентств, который автоматизирует работу с инфлюенсерами и помогает растить охваты на международных рынках.

## Сегменты (гипотеза)

*Не валидировано. После первой серии AJTBD-интервью заполнится в `wiki/segments/`.*

Кандидатные направления:
- **B2B международные DTC-бренды** — нужны охваты через инфлюенсеров без раздутого in-house маркетинга.
- **Агентства инфлюенс-маркетинга** — нужны AI-инструменты для масштабирования операционки.
- **SMB / селлеры на маркетплейсах** — нужны дешёвые AI-генерируемые креативы для инфлюенсеров.

См. отдельные страницы `wiki/segments/*.md` (пока пустые).

1. Провести 5 AJTBD-интервью → положить транскрипты в `raw/interviews/`.
2. Ingestнуть каждое, наполнить `wiki/segments/`.
3. Сделать `competitive-brief` по 3 ключевым конкурентам.
4. Зафайлить ADR "выбранный фокус-сегмент" в `wiki/decisions/`.
5. Параллельно: начать `lauda-web` (Cursor) — landing-MVP под выбранный сегмент.
