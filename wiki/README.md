# wiki/ — синтезированное знание

**LLM-агент владеет этой папкой целиком.** Люди только читают и просят.

Не редактируй файлы здесь руками — это приведёт к рассинхрону с источниками. Если нужно поправить — попроси LLM сделать обновление с указанием причины. Запись попадёт в `log.md`.

## Структура

- `index.md` — каталог всех страниц. Начинай отсюда.
- `log.md` — хронология действий (ingest / query / lint).
- `overview.md` — высокоуровневая картина бизнеса.
- `concepts/` — общие концепты.
- `segments/` — клиентские сегменты по AJTBD.
- `competitors/` — страницы конкурентов.
- `customers/` — карточки реальных клиентов / интервьюируемых.
- `product/` — продуктовые решения, фичи, гипотезы.
- `sales/` — pipeline, сделки.
- `marketing/` — каналы, кампании.
- `team/` — команда, найм.
- `finance/` — метрики, юнит-экономика.
- `decisions/` — ADR (Architecture/Strategic Decision Records).
- `sources/` — summary каждого raw-источника.
- `comparisons/` — сравнения.
- `_templates/` — шаблоны страниц (для LLM, не для людей).

См. конвенции в [`../CLAUDE.md`](../CLAUDE.md).
