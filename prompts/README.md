# prompts/

Переиспользуемые промпты для команды и LLM-агентов. Каждый промпт — отдельный `.md` файл. Используй как: `Примени промпт prompts/<name>.md к <данным>`.

## Доступные промпты

- `ingest.md` — стандартный workflow ingest-а источника.
- `weekly-lint.md` — еженедельный health-check wiki.
- `competitor-research.md` — добавить нового конкурента в wiki.
- `interview-debrief.md` — разбор AJTBD-интервью.
